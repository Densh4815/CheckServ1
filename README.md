# 🚀 CheckServ1 - Telegram Site Monitor Bot

Простой и надежный Telegram-бот для мониторинга доступности сайта с уведомлениями о сбоях и восстановлении работы.

## ✨ Возможности

* 🌐 Проверка доступности сайта 24/7
* 🚨 Уведомления при недоступности сайта
* ✅ Уведомления о восстановлении работы
* 🔕 Защита от ложных срабатываний
* 📊 Статистика мониторинга
* 👥 Подписка на уведомления через Telegram
* ⚡ Легкий и быстрый запуск

---

## 🤖 Команды бота

| Команда        | Описание                      |
| -------------- | ----------------------------- |
| `/start`       | 🚀 Начало работы              |
| `/status`      | 📊 Текущий статус сайта       |
| `/stats`       | 📈 Статистика мониторинга     |
| `/subscribe`   | 🔔 Подписаться на уведомления |
| `/unsubscribe` | 🔕 Отписаться от уведомлений  |
| `/help`        | ℹ️ Справка                    |

## ⚙️ Установка

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/Densh4815/CheckServ1.git
cd CheckServ1
```

### 2️⃣ Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3️⃣ Настройка токена Telegram

Создайте переменную окружения:

```bash
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

---

## ▶️ Запуск

```bash
python main.py
```

---

## 🔧 Основные настройки

В файле `main.py`:

```python
CHECK_URL = "https://example.com/"
CHECK_INTERVAL = 10
MAX_CONSECUTIVE_ERRORS = 3
```

### Описание параметров

* 🌐 `CHECK_URL` — адрес сайта для мониторинга
* ⏱️ `CHECK_INTERVAL` — интервал проверки в секундах
* 🚨 `MAX_CONSECUTIVE_ERRORS` — количество ошибок подряд для фиксации сбоя

---

## 📋 Логика работы

1. 🔄 Бот проверяет сайт через заданный интервал.
2. ❌ При нескольких ошибках подряд фиксируется сбой.
3. 🚨 Отправляется уведомление подписчикам.
4. ✅ После восстановления сайта отправляется уведомление о восстановлении.
5. 🔕 Повторные уведомления не отправляются до следующего сбоя.
